# ☕ Golden Bean – Coffee Table Reservation App (UI/UX)

A modern and elegant **Coffee Table Reservation Mobile App** designed in **Figma**. Golden Bean offers customers a smooth and hassle-free experience for reserving tables, selecting preferred time slots, and completing secure payments.

## 📱 Project Overview

Golden Bean is a mobile-first UI/UX design created to simplify café table reservations. The app features a premium **Black & Gold** theme, intuitive navigation, and a seamless booking process that enhances the customer experience.

## ✨ Features

- ☕ Beautiful Splash Screen
- 🔐 Secure Login & Sign Up
- 🪑 Table Selection
- 📅 Date & Time Selection
- 📝 Booking Review
- 💳 Secure Payment
- ✅ Booking Confirmation
- 📱 Mobile-First Responsive Design
- 🎨 Modern & Minimal User Interface
