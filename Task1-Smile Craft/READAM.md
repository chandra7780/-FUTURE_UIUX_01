This project was created as part of the Future Interns UI/UX Design Internship Program.

## Project Overview

Smile Craft is a modern dental appointment booking mobile application designed to provide users with a clean and easy booking experience.

The application allows users to:

* Login securely
* Browse dental services
* View treatment pricing
* Select appointment date & time
* Confirm appointments
